# 简历 Skill 安装与使用说明

## Skill 目录

本仓库包含两个可配合使用的 skill：

- `resume_latex_skill/`
- `resume_layout_optimizer_skill/`

## 推荐输入方式

优先让用户填写：

- `input_templates/resume_input_template.md`

如果用户更习惯 Word，可先填写：

- `input_templates/resume_input_template_word.docx`

再运行 `resume_latex_skill/scripts/word_to_markdown.py` 转为 Markdown。

## Agent 使用方式

当 Agent 需要根据 Markdown 或整理后的 Word 内容生成 LaTeX 简历时，触发：

- `resume-latex-builder`

当 Agent 只需要做排版诊断、压字、单页优化或间距调整时，触发：

- `resume-layout-optimizer`

## 推荐流程

1. 检查依赖
2. 如果输入是 Word，先转 Markdown
3. 规范编码
4. 运行 `scripts/render_resume.py` 生成 `main.tex`
5. 串行编译
6. 运行 `resume_layout_optimizer_skill/scripts/analyze_layout.py` 输出诊断报告

也可以直接运行总控脚本：

- Markdown 输入：`resume_latex_skill/scripts/build_from_md.ps1`
- Word 输入：`resume_latex_skill/scripts/build_from_word.ps1`

## 当前能力边界

- 已支持：Markdown -> TeX -> PDF
- 推荐路径：Word -> Markdown -> TeX -> PDF
- 已支持：嵌套多层 bullet
- 已支持：标题超链接 `title_link`
- 已支持：编译后排版诊断
